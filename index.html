Troubleshooting Guide:

  • Nothing happens when clicking: Ensure you're on a Shopify admin page and your browser is up to date.
  • Details don't hide: Shopify may have updated their UI. Check for a newer version of this tool.
  • Error message appears: Try refreshing the page and clicking again.
  • Bookmark disappeared: Some browsers hide bookmarks. Right-click the bookmarks bar and ensure "Show bookmarks bar" is enabled.

Common Use Cases:

  • Order Processing: Hide unnecessary details while processing multiple orders
  • Printing Orders: Create cleaner printouts without subdued text
